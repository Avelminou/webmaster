Introduction à HTML


      HTML signifie HyperText Markup Language. C’est le langage utilisé pour créer la structure des pages web.


Un fichier HTML se termine par ".html"



Exemple simple



<!DOCTYPE html>
<html lang="fr">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mon site</title>
</head>

<body>

    <h1>Bonjour étudiants</h1>
    <p>Apprendre HTML est facile.</p>

</body>
</html>


Explication


  • <!DOCTYPE html> → indique que c'est du HTML5
  • <html> → racine du document
  • <head> → informations de la page
  • <body> → contenu visible
  • <h1> → Un grand titre
  • <p> → paragraphe


Les titres


HTML possède 6 niveaux de titres. 

<h1>Titre principal</h1>
<h2>Sous titre</h2>
<h3>Sous section</h3>
<h4>Titre petit</h4>
<h5>Titre petit</h5>
<h6>Titre petit</h6>
     h1 est leplus grand.
     h6 est leplus petit.


Les paragraphes


     Pour écrire du texte :

<p>Ceci est un paragraphe.</p>
<p>Un autre paragraphe.</p>

Saut de ligne



<br>

Ligne horizontale



<hr>

Eemple:


<p>Bonjour!</p>
<hr>
<p>Je suis un paragraphe.</p>
<br>
<p>Au revoir!</p>

Je suis un paragraphe.

Les Attributs HTML


     Un attribut donne une information à une balise.

Exemple:



<div class="ma-page-gauche"></div>

class est un attribut.

Autres attributs :
  • id
  • href
  • style
  • src
  • alt
